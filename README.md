# Mahmoud Shayeb — Portfolio

**One portfolio. Five ways to see it.**

You land here once—then pick your lens: Developer, Recruiter, Founder, Technical Lead, or Just Curious. The same person, the same work, framed for who you are. No generic “about” page; the content and tone shift to match why you’re here.

---

## Tech stack

- **React** · **TypeScript** · **Vite**
- **Tailwind CSS** · **Framer Motion**
- **Zustand** · **React Router**

---

## Coming soon

- **Blog** — Articles and notes, with markdown and a proper feed.
- **Calendly** — Book a call or meeting directly from the site.
- **CLI (Developer)** — Terminal-style bar: run commands like `/about`, `/projects` for a dev-first experience.
- **Debug Me (Developer)** — A small, interactive “debug this” challenge.
- **Timeline & CV (Recruiter)** — Career timeline view and one-click CV download.
- **Flow diagrams (Technical Lead)** — Architecture and decision-flow visuals.
- **SaaS demo (Founder)** — A short, product-style walkthrough.
- **Games & easter eggs (Just Curious)** — Lightweight games and hidden bits for explorers.
- **AI chat** — Persona-aware chat (e.g. technical for developers, impact-focused for recruiters) once wired to a backend.

More is in the pipeline. Watch this space.

## Run locally

```bash
yarn install
yarn dev
```

Build: `yarn build` · Preview: `yarn preview`
